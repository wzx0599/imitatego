# Imitate-Go

## 简介

学习和模仿别人的代码，用C++实现典型的算法，并编写相应的测试用例。

## 目前已完成

+ 排序算法
  + 快速排序 ```quick_sort.cpp```
  + 选择排序 ```selection_sort.cpp```
  + 归并排序 ```merge_sort.cpp```
  + 堆排序 ```max_heap.cpp``` or ```priority_queue.cpp```
+ LRU缓存算法 ```lru_cache.cpp```
+ AVL树 ```avl_tree.cpp```
